## Quick Start
1. **Install Dependencies**: `npm install`
2. **Setup Environment**: Copy `documentation/env.example` to `.env` and fill in credentials.
3. **Setup Database**: `npx prisma generate` && `npx prisma db push`
4. **Run**: `npm run dev`
